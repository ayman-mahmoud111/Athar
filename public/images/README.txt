ATHAR Website Images
====================

This folder is for storing product images and other website assets.

How to add images:
1. Add your images here with descriptive names (e.g., product-1.jpg, hero-bg.png)
2. Reference them in your code as: src="/images/filename.jpg"

Recommended image sizes:
- Product images: 800x1000px (portrait) or 1000x800px (landscape)
- Hero/banner images: 1920x1080px
- Thumbnail images: 400x400px
- Category images: 600x600px

Supported formats: JPG, PNG, WebP, GIF

مجلد صور موقع أثر
================

هذا المجلد مخصص لحفظ صور المنتجات والأصول الأخرى للموقع.

طريقة إضافة الصور:
1. ضع الصور هنا بأسماء واضحة (مثال: product-1.jpg, hero-bg.png)
2. استدعها في الكود كالتالي: src="/images/filename.jpg"

أحجام الصور الموصى بها:
- صور المنتجات: 800x1000px أو 1000x800px
- صور الهيرو/البانر: 1920x1080px
- الصور المصغرة: 400x400px
- صور الأقسام: 600x600px

الصيغ المدعومة: JPG, PNG, WebP, GIF
